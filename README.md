# Todo List Application

## Overview
This is a modern and beautiful Todo List application built using HTML, CSS, and Vanilla JavaScript. The application allows users to manage their tasks efficiently with features such as adding, editing, deleting, and filtering tasks.

## Features
- **Add Tasks**: Easily add new tasks to your list.
- **Edit Tasks**: Modify existing tasks with a simple edit mode.
- **Delete Tasks**: Remove tasks after confirmation.
- **Complete Tasks**: Mark tasks as completed or active.
- **Filter Tasks**: Filter tasks based on their status (all, active, completed).
- **Search Tasks**: Real-time search functionality to find tasks quickly.
- **Sort Tasks**: Sort tasks based on different criteria.
- **Statistics**: View total tasks and completed tasks counters.
- **Clear Completed Tasks**: Remove all completed tasks with a single click.
- **Responsive Design**: Works seamlessly on various devices.

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## Setup Instructions
1. Clone the repository or download the project files.
2. Open `index.html` in your web browser to view the application.
3. You can modify the `style.css` file to change the appearance of the application.
4. The `script.js` file contains all the JavaScript logic for the application.

## Usage Guidelines
- To add a task, type in the input field and press Enter or click the Add button.
- To edit a task, click on the task text, make your changes, and press Enter.
- To delete a task, click the delete button next to the task.
- Use the filter buttons to view all tasks, only active tasks, or only completed tasks.
- Use the search bar to find specific tasks quickly.
- Sort tasks using the dropdown menu to organize them as needed.
